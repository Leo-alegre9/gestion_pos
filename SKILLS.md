# SKILLS.md — Sistema POS para Bar

## 🎯 Propósito del proyecto

Desarrollar un sistema POS (Point of Sale) para un bar con una sola sucursal, enfocado en:
- Gestión de mesas
- Toma de pedidos
- Cobros y caja
- Control de stock
- Administración de usuarios
- Reportes operativos

## 📌 Stack del sistema
- Framework: CodeIgniter 4
- Lenguaje: PHP 8+
- Base de datos: MariaDB (XAMPP local) y se implementará en un hosting a futuro
- Frontend: Bootstrap 5 + Animate.css
- Arquitectura: MVC + Services

## 🧠 Skills necesarias
### Backend
- Controllers, Models, Views
- Filters, Validation, Sessions
- Services, Migrations, Seeders
- Buenas prácticas SOLID

### Base de Datos
- Modelado relacional
- PK/FK
- JOIN, GROUP BY, SUM, COUNT
- Transacciones
- Índices

### Frontend
- Bootstrap Grid
- Forms
- Modals
- Tables responsive
- UX para pantallas táctiles

### Seguridad
- Login / Logout
- Roles y permisos
- Hash de contraseñas
- CSRF / XSS

### Módulos
- Login
- Dashboard
- Mesas
- Pedidos
- Caja
- Productos
- Stock
- Usuarios
- Reportes

## Regla principal
Priorizar velocidad operativa, simplicidad y confiabilidad. Sistema minimalista, enfocarse en el módulo de pagos y pedidos. Documentar cada una de las funciones implementadas con comentarios inline explicativos tanto en el frontend como en el backend.
